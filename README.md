# База данных PostgreSQL для сервиса по доставке цветов.


```flowers_delivery_schema.sql``` - схема базы данных, добавление данных в таблицы;

```flowers_delivery_queries.sql``` - запросы к базе данных и (не)материализованные представления.

Бизнес-логика:


В интернет-магазине продаются букеты из цветов. Каждый цветок имеет название и стоимость за 1 шт. Каждый букет имеет в себе несколько разных цветков разного количества. 
Пользователь регистрируется на сайте. Он указывает номер телефона, имя, почту, дату рождения (необязательно), город. Он может сформировать один или несколько заказов. Каждый заказ включает в себя только букеты. Также пользователь может указать к заказу комментарий. После формирования заказа, он проходит ряд этапов (оплачивается, собирается, отдается курьеру, едет, доставляется) - каждому этапу присвоена временная метка.
При этом есть склад. На складе ведется учет товаров. Складов несколько, каждый в своем городе.
